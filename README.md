# Yosif Al-khazaali - CS 499 ePortfolio

This repository is for my CS 499 Computer Science Capstone ePortfolio.

The purpose of this portfolio is to show the work I have completed throughout my Computer Science program and how I can improve that work into something more complete, organized, and professional. This is not just a place to drop old assignments. The main goal is to take selected artifacts from previous courses, enhance them, explain the changes I made, and show how those changes connect to real computer science skills.

## About This Portfolio

My ePortfolio will focus on three main areas:

- Software Design and Engineering
- Algorithms and Data Structures
- Databases

Throughout the capstone, I will be adding my original artifacts, enhanced versions, written narratives, and a code review. I will also include a professional self-assessment that explains my growth through the Computer Science program and how these projects connect to my career goals.
