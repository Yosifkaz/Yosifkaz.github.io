About This Portfolio

This ePortfolio is being created for my CS 499 Computer Science Capstone. The purpose of this site is to bring together selected work from my Computer Science program, improve it, and explain the decisions behind those improvements. I want this portfolio to show more than just finished assignments. I want it to show how I think through software problems, how I improve my code, and how I connect what I learned in the program to real technical work.

Selected Artifact Areas

For this capstone, I will be choosing artifacts that show growth in the main areas required for the course. These areas are software design and engineering, algorithms and data structures, and databases.

Software Design and Engineering

This section will focus on improving the structure, usability, reliability, and overall design of a software project.

Algorithms and Data Structures

This section will focus on improving logic, efficiency, organization, and the use of data structures to solve a problem.

Databases

This section will focus on how data is stored, searched, managed, protected, and used by an application.

Planned Enhancements

As I move through the capstone, I will add my original artifacts, enhanced versions, written narratives, and my code review. Each enhancement will explain what the original project did, what I changed, why I made those changes, and what skills the changes demonstrate.
